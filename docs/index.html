Habit Tracker Box - Code Download

* If you are tracking fewer than five habits, leave the remaining habit name fields blank. Please fill them out in order from top to bottom.

* Option named "Once Is Enough" will allow you to set one or two days in a week that a goal should be completed by. The associated button

should be pressed before the designated goal days or the display will say "__ days late".

* Option named "Weekly Quota" will allow you to set one day in a week that a goal can be completed by as many times as you decide. You have

the choice to set the number of times you want to complete the goal in a week. The system will automatically reset the counter at the end

of the week.
